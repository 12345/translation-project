Typesafe Activator とは?

Typesafe Activator は、Play Framework, Akka そして Scala を含む Typesafe Reactive プラットフォームです。 これは、リアクティブなアプリケーションを作成したい開発者に向けた、新しいアプリケーションを作成するためのブラウザ内またはコマンドライン環境を提供する活動拠点です。

もっとも早く Play を始める方法

はじめるのは簡単です! Activator をダウンロードして、展開、実行すると、これから始めるテンプレートを選択するようウィザードベースのインタフェースがガイドしてくれます。 どのテンプレートにも、それぞれの状況に応じたチュートリアルとコードの概要が含まれています。

チュートリアルを含む人気のテンプレート:

  • Hello Play Framework!
  • Play with Guice
  • Play Reactive Mongo with KnockoutJS frontend
  • Play + Java + Spring
すべてのテンプレートを見る

Typesafe Activator を実行する

Typesafe Activator をダウンロードして展開する

Activator ${activator.version} (${activator.size}) をダウンロードします。

ダウンロードした zip ファイルをシステムに展開します。

Typesafe Activator の UI を開始する

ファインダで Typesafe Activator を展開したディレクトリまで移動したら、activator という名前のファイルを右クリックして、"Open" を選択します。警告が表示されたら続行をクリックします。

またはコマンドラインから実行します:

 $ activator-${activator.version}/activator ui

こうすると Typesafe Activator が開始し、ブラウザを開いて Activator UI を表示します。

新しいアプリケーションを作成する

作成するアプリケーションを 多数のアプリケーションテンプレート から選びます。

アプリケーションが作成されれば、コードを探索し、テストを実行し、アプリケーションを実行することができます。どのテンプレートにも、アプリケーションの個々のパーツと、それらがどのようにして動作するかについて段階的に説明するチュートリアルが含まれています。

従来の Zip を使って始めよう
