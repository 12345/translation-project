Play と Activator

Activator で Play をインストールして実行する

コマンドラインにおける Play

cli でコマンドが使えるよう Activator を PATH に追加 してください。

コマンドラインで新しいプロジェクトを作ります:

activator new

既存のプロジェクトを、そのディレクトリ内から実行します:

activator run

(プロジェクトディレクトリ内で) インタラクティブな cli を起動します:

activator

Activator の ui とチュートリアル

学習、コンパイル、テスト、実行

Activator には Play を学習する ui が備わっています。実行可能な activator をダブルクリックするか、またはターミナルから以下のコマンドを実行して、ui を立ち上げてください:

activator ui

ブラウザに Web ページが表示されるので、選択した説明に従って、チュートリアルを開いてください。

より詳しくは、Play 向けの Activator チュートリアル をご覧ください。

Activator についてもっと詳しく

Typesafe Activator で Play と Typesafe リアクティブプラットフォーム を始めましょう。Activator は、リアクティブアプリケーションを作成したいと考える技術者のハブであり、新しいアプリケーションを作成するコマンドラインおよびブラウザ環境を提供します。

はじめるのは簡単です。紹介動画を含め、ひと通りのものが揃った はじめよう ページの説明を参照してください。
