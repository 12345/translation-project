プロフェッショナルサービスとホスティング

コンサルティング、トレーニングとサポート

Play framework のバックエンドである Zenexity は、Play framework の全ての威力を解き放つために、その主要概念を理解するお手伝いをすることができます。

私たちはあなたの Play アプリケーションを監査し、あなたのチームを牽引し、あなたのプロジェクトを立ち上げ、あなたの既存情報システムへ Play framework を統合します。私たちは Play を使ったミッションクリティカルなアプリケーションのプロフェッショナルサポートを提案することもできます。

Lunatech Research, Netherlands

Lunatech は Play の初期から貢献してきました。 私たちは、エレガントなアーキテクチャ設計、既存サービスの再利用、磐石な RESTful web サービスや web アプリケーションの構築をお手伝いすることができます。

もしあなたの Play アプリケーションを既存の Java EE サービススタックに統合したいけれど、どのようにしたらよいか分からない場合、私たちにお声掛けください。 私たちは技術トレーニングも提供しています。私たちは、開発者の質問、フレームワークのバグ修正と改良について合意した SLA に基づく第三者サポートを提供することもできます。

ホスティング

playapps.net

Playapps.net は、あなたの Play アプリケーションを素早く効率的に起動し実行するよう設計された合理的なデプロイ環境です。

Playapps.net は、Play framework のシンプルさを念頭において構築されており、あなたが自前のサーバでアプリケーションを実行するために苦労せずに済むように、各スロットにはあなたのアプリケーションを実行するために必要なすべてのサービスが含まれています。
