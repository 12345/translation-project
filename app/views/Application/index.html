Play Framework で Java と Scala の web アプリケーション開発を
もっと簡単に

Play は web と親和性の高い軽量でステートレスなアーキテクチャに基づいています。

Akka を基盤とすることで、Play は高度にスケール可能なアプリケーションのための予測可能で最小限のリソース (CPU、メモリ、スレッド) 消費を提供します。

開発者に優しい

変更したら更新ボタンをただ叩く! 必要なのはブラウザとテキストエディタだけ。

  • 更新ボタン押下ワークフロー
  • 強力なコンソールとビルドツール
  • 型安全
  • 組み込みテストツール
  • Eclipse と IntelliJ IDEA のサポート
Play instances usage

期待通りにスケール

Play は Akka 上に構築された完全な非同期モデルを内部で使用します。ステートレスであることと組み合わさり、Play は単純かつ期待通りにスケールします。

  • ステートレスな Web 層
  • ノンブロッキング I/O
  • Akka ベース
  • リアルタイム可能

近代的な web とモバイル

Play は近代的な web およびモバイルアプリケーション向けに作られています。

  • デフォルトで RESTful
  • CoffeeScript, LESS, その他用のアセットコンパイラ
  • ファーストクラスとしての JSON
  • Websockets, Comet, EventSource
  • NoSQL とビッグデータの広範囲なサポート

堅実かつ高速

コンパイラと実行環境 (JVM) が力仕事をしてくれるため、アプリケーションは超高速に動作し、負荷状態でも動き続けます。

  • コンパイルされたコード
  • JVM 上で実行
  • Netty による Java NIO

巨大なエコシステム

Java 周辺のエコシステムは巨大です。ありとあらゆるライブラリがあり - そのほとんどが Play で利用できます。

  • Maven セントラルライブラリを利用
  • 多数の Play プラグイン
  • 活発な Play コミュニティ

本番環境での実績

スタートアップからエンタープライズまで、Play はとても革新的な web サイトのいくつかに役立っています。Play には実績、信頼性、そして拡張性があります。

  • Typesafe による本番サポート
  • クラウド環境へのデプロイオプション
  • ミッションクリティカルなアプリケーションでの利用
