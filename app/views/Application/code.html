コードについて

Play Framework は Git バー所管理システムを使って Github で管理されています。次のコマンドでコードを取得することができます:

git clone git://github.com/playframework/Play20.git

以前の Play 1.x シリーズのコードを探している場合は、別のリポジトリから利用することができます:

git clone git://github.com/playframework/play.git

貢献するときは 貢献者ガイド を調べて、そして Play をソースからビルドする 方法を学んでください。

セキュリティレポート

セキュリティアップデートを受け取る

あらゆるすべてのセキュリティ報告を受け取る最善の方法は、 Play security list に登録することです。

このメーリングリストの流量はとても少なく、セキュリティレポートがコアチームによって管理され、修正版が公開されたあとにだけ、通知が投稿されます。

Play Framework のセキュリティ的な問題を見つけた場合は?

セキュリティ的な問題は、公開されたフォーラムで発表する前に、まず非公開のセキュリティメーリングリストに報告することを強く推奨します。

すべてのセキュリティに関するバグは e メールにて security@playframework.org に報告されるべきです。 このリストは、セキュリティ的な問題を扱うコアチームの一部に配信されます。
