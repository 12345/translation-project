Play 2.1.1

2013/04/13 リリース

  • multipart/form-data API に対する変更を切り戻し、ファイルが何も送信されない状態を容易に検出できるようになりました
  • リバースルーターがパス引数の文字列をエスケープするようになりました
  • クエリ文字列とパスにおける不正なエスケープの組み合わせは 400 エラーを返すようになりました
  • includes のみを含む routes ファイルもコンパイル対象になりました
  • Javadoc 生成処理が動作するようになりました
  • コンパイルされたアセットがソース監視対象ではなくなりました
  • Windows におけるコンパイルに関するいくつかの問題を修正
  • routes ヘルパーの実装を改善
  • 884 2GB 以上のファイルアップロードが再び可能に (これは Netty の制限であり、3.6.3 にアップグレードすることで解消しました)
  • 889 Scala における JSON シリアライズのパフォーマンス/平行処理性能に関するボトルネックを解消
  • 888 Javascript リバースルーターのデフォルト値を修正
  • JSON マクロの修正と改善
