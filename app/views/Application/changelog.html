アップグレードに関する注意

  • CSRFFilter コンストラクタのシグネチャが変更されました。以前のものを使っているコードは依然としてソース互換性がありますが、リコンパイルする必要があるかもしれません。
  • JavaScript コンパイラを使う際、CommonJS モジュールのサポートがデフォルトで提供されていました。CommonJS の Windows サポートにおいて問題があるため、今後はデフォルトでは提供しません。CommonJS モジュールのサポートは JavaScript コンパイラに commonJs オプションを与えることで再度有効化することができます。
